# synapse3d
Lightweight WebGL-based 3D editor for fast modeling, prototyping, and AI-oriented workflows

## Demo
https://kyoto-hirota.jp/synapse3d/

## Features
- Lightweight browser-based 3D editor
- Primitive generation
- Face selection & docking
- Geometry merge
- STL export
- JSON save/load
- History system with undo
